# Blinkit-Sales-Dashboard 📊
A dynamic and interactive Blinkit Sales Dashboard created using Microsoft Excel to analyze customer behavior, order trends, and regional sales performance. This dashboard helps visualize key business metrics such as total orders, customer segmentation, average order value, and registration trends.

📌 Project Overview

This dashboard was built to perform sales analysis on Blinkit-style retail/order data and present insights in a clean and visually interactive format using:

Pivot Tables
Pivot Charts
Slicers
KPI Cards
Excel Dashboard Design Techniques

The project focuses on understanding:

Customer segments
Regional order performance
Average order value
Customer registrations over time
